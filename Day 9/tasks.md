# Task: Compare Two Machine Learning Classification Models

## Objective

Train and evaluate two different machine learning classification models using the dataset assigned to you. Compare their performance using confusion matrices and standard classification metrics.

---

## Requirements

### 1. Data Preparation

- Load the assigned dataset.
- Perform any necessary preprocessing.
- Split the dataset into training and testing sets.

---

### 2. Model Selection

- Choose two different classification algorithms.
- Train both models using the same training data.

---

### 3. Model Evaluation

For each model, generate and include:

- Confusion Matrix
- Accuracy
- Precision
- Recall
- F1-Score
- Classification Report

---

### 4. Model Comparison

Compare the performance of both models based on:

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

Provide a brief explanation of which model performed better and why.

---

## Deliverables

Submit the following:

- Source code
- Confusion Matrix for Model 1
- Confusion Matrix for Model 2
- Classification Report for both models
- Comparison of evaluation metrics
- A brief conclusion summarizing the results
